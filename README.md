Finance Wrapped is a personal finance insights project inspired by Spotify Wrapped.
It transforms raw bank statements into easy to understand spending insights, trends,
and summaries making personal finance more engaging for Gen Z.

---

## Why this project?

Most people avoid tracking finances because:
- Bank statements are boring
- Spreadsheets feel overwhelming
- Insights are not visual or intuitive

Finance Wrapped aims to change that by presenting financial data in a fun,
story like, and visual way similar to Spotify Wrapped.

---

## Current Features (Backend)

- Express.js backend setup
- CSV file upload API
- CSV parsing into structured JSON
- Clean route service separation
- Ready foundation for analytics and insights

---

## Tech Stack

- Node.js
- Express.js
- Multer (file uploads)
- csv-parser
